# WhatsApp Quick Reach — My WA Tool

An installable **Progressive Web App (PWA)** for sending pre-saved WhatsApp message templates with a single tap — built for sales and outreach reps who send the same messages repeatedly.

## What It Does

Stores your most-used WhatsApp message templates and opens them directly in WhatsApp with one tap — eliminating the need to copy-paste messages every time. Works offline and installs like a native app.

## Key Features

- **One-tap template send** — tap any template to open WhatsApp with the message pre-filled
- **Saved message templates** — store all your frequently-used WhatsApp messages in one place
- **Installable PWA** — add to home screen on Android/iOS, works in standalone display mode
- **Offline-capable** — service worker (`sw.js`) caches the app for use without internet
- **Portrait mode optimized** — designed for phone use
- **WhatsApp brand theming** — familiar green (#075e54) UI
- **No login or account needed** — open and use immediately
- **Custom icons** — branded app icon for home screen

## Tech Stack

- HTML, Vanilla JavaScript
- PWA (manifest.json + service worker `sw.js`)

## How to Use

1. Open the app in your mobile browser
2. Tap "Add to Home Screen" to install
3. Tap any template to open WhatsApp with the message ready
4. Hit send

## Audience

Sales reps, outreach teams, and anyone who sends repeated WhatsApp messages daily.
